# Математический квиз

Интерактивная викторина по математике 10-11кл

![React](https://img.shields.io/badge/React-18.2.0-blue)
![TypeScript](https://img.shields.io/badge/TypeScript-5.0.0-blue)
![License](https://img.shields.io/badge/License-MIT-green)

## Возможности

-  10 математических вопросов
-  3 уровня сложности
-  Система начисления баллов
-  Таблица лидеров (сохраняется в браузере)
-  Перемешивание вариантов ответов
-  Мгновенная проверка ответов
-  Адаптивная верстка

## Быстрый старт

### Требования

Для запуска проекта необходим [Node.js](https://nodejs.org/) (версия 18 или выше)

### Установка и запуск

```bash
# 1. Клонируйте репозиторий
git clone https://github.com/DiavolIII/math-quiz.git

# 2. Перейдите в папку проекта
cd math-quiz

# 3. Установите зависимости
npm install

# 4. Запустите проект
npm start
```
